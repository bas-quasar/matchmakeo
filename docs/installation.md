## Installation

Install matchmakeo using `pip` from the source on GitHub, we recommend doing so in a virtual environment such as venv or conda, or using a tool such as `uv`.

### with `pip`

over https
```sh
pip install git+https://github.com/bas-quasar/matchmakeo.git
```

or over ssh

```sh
pip install git+ssh://git@github.com/bas-quasar/matchmakeo.git
```


### with `uv`

over https
```sh
uv add git+https://github.com/bas-quasar/matchmakeo
```

or over ssh
```sh
uv add git+ssh://git@github.com/bas-quasar/matchmakeo.git
```
